# fluidsim
Simple Fluid Simulation Visualizer

This is a simple fluid simulation that allows you to create sources and sinks as well as barriers. You have the ability to drop particles in an area of the simulation and see how they evolve over time. 

There are a variety of property visualizations. 
- Color
- Kinetic Energy
- Strain
- Vorticity
- Pressure
- Thermal
- Velocity
- Monochrome

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
